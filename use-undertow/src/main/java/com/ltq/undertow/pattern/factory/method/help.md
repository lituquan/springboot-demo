### 对比静态工厂
        静态工厂通过条件获取产品，当需要增加产品类型时，需要修改工厂

        将工厂进行抽象，就可以不入侵工厂代码。而是通过增加产品和它的模板工厂，来获取对象。

        例如：
         增加a^b计算,静态工厂需要修改代码。但是工厂方法是增加一个工厂即可。
